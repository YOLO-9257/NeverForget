# Security Policy

## Supported Versions

The following versions of CF-Reminder are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you've found a security vulnerability in CF-Reminder, please DO NOT open a public issue. Instead, please report it via one of the following methods:

- **Email**: ying@linux.do
- **GitHub**: Use the "Report a vulnerability" button on the Security tab of the repository.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please keep the details private until a fix is released.

Thank you for helping keep CF-Reminder secure!
